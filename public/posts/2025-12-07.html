# Personal news

One nice thing about my current apartment is its small outdoor balcony. I set up a bird feeder on the balcony in late November, wondering if it would help the local bird population out during the winter.

# Website news

Added a "Table of Contents" feature for my blog posts this weekend! My blog posts can run pretty long, especially longer reflections on PhD milestones or research conference travel. A "Table of Contents" feature seemed like a nice way to improve navigation through longer blog posts :)

Table of Contents plugins:

  • markdown-it-anchor: "A markdown-it plugin that adds an id attribute to headings and optionally permalinks"
  • eleventy-plugin-nesting-toc: "generates a (property nested) TOC from page content using an Eleventy filter"

Still tweaking the styling (and wondering whether I want the anchor signs visible -- but the ToC broke when I tried to hide them TT_TT), but I like how the feature works so far! I'm still deciding whether I'll use it on every blog post going forward, or only on particularly long ones. Either way, I've added it to this post just so users can test it out :)
