mayworms.info is built in <a href="https://www.11ty.dev" target="_blank"><b>11ty (Eleventy)</b></a>, and is hosted on <a href="https://docs.github.com/en/pages" target="_blank"><b>Github Pages</b></a>. Prior to the 2024 redesign, mayworms.info was built in <a href="https://jekyllrb.com" target="_blank"><b>Jekyll</b></a> and used the <a href="https://mmistakes.github.io/minimal-mistakes/about/" target="_blank"><b>Minimal Mistakes</b></a> Jekyll theme. 

Below is a <b>list of plugins</b> I use on mayworms.info. I also included some <b>tutorials and guides</b> (for both Eleventy and for HTML/CSS/Javascript generally) that I leaned on while rewriting my website. I hope something here is helpful for you! (˶ᵔ ᵕ ᵔ˶)

---

### Plugins
<a href="https://github.com/feimosi/baguetteBox.js" target="_blank"><b>baguetteBox.js</b></a>: I use this pure JavaScript lightbox plugin for this website's image galleries! 

<a href="https://github.com/paulirish/lite-youtube-embed" target="_blank"><b>Lite YouTube Embed</b></a>: This plugin helps embedded YouTube links render a bit faster :)

<a href="https://github.com/11ty/eleventy-plugin-font-awesome" target="_blank"><b>@11ty/font-awesome</b></a>: The official FontAwesome plugin for Eleventy! I use this plugin for my Contact list icons. 

<a href="https://www.npmjs.com/package/markdown-it-anchor" target="_blank"><b>markdown-it-anchor</b></a>: I use this plugin to add an "id" attribute and permalinks to my website's headings (see those little hashtags next to this page's headings? That's why they're there!). This plugin is strongly recommended if you also plan to install the <b>Table of Contents (nesting) Eleventy Plugin.</b>

<a href="https://github.com/JordanShurmer/eleventy-plugin-nesting-toc" target="_blank"><b>Table of Contents (nesting) Eleventy Plugin</b></a>: This plugin is used for the "Table of Contents" button on long blog posts!

### Tutorials

#### Eleventy
<a href="https://renkotsuban.com/posts/2025-07-25-Updates-to-the-Eleventy-guide.html" target="_blank"><b>Renkon's Eleventy Guide</b></a>: Easily the best "getting started with Eleventy" resource I found while rebuilding mayworms.info. My website's overall design, particularly its blog structure, is based heavily on Renkon's tutorial. 

<a href="https://renkotsuban.com/posts/2025-07-26-Eleventy-plugin-recs-part-1.html" target="_blank"><b>Renkon's Eleventy Plugin Recs (Pt. 1)</b></a>: If you found Renkon's Eleventy starter guide helpful, I strongly recommend you explore their Eleventy plugin recommendations as well! 

<a href="https://equk.co.uk/2025/03/23/11ty-font-awesome-svg-plugin/" target="_blank"><b>EQUK: 11ty Font Awesome SVG Plugin</b></a>: This tutorial taught me how to install the FontAwesome Eleventy plugin (and how to style FontAwesome's SVG icons correctly). Now I don't have to use emoji in my Contact list anymore!

#### CSS
<a href="https://developer.mozilla.org/en-US/docs/Web/CSS" target="_blank"><b>MDN Web Docs</b></a>: I had very little CSS background when I redesigned mayworms.info, so I wound up spending hours on MDN Web Docs learning how flexboxes work. Very helpful tutorials!

<a href="https://www.w3schools.com/css/" target="_blank"><b>W3Schools</b></a>: W3Schools is the resource I used in high school while first learning basic HTML, CSS, etc -- still very helpful today!
